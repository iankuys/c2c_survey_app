# c2c-discrete-choice-video

Written with Python 3.11.3.

Set up a virtual environment for developing this app (example with PowerShell on Windows 11):
```
PS C:\...\c2c-discrete-choice-video> python -m venv .venv
PS C:\...\c2c-discrete-choice-video> .\.venv\Scripts\Activate.ps1
(.venv) PS C:\...\c2c-discrete-choice-video> python -m pip install --upgrade pip
(.venv) PS C:\...\c2c-discrete-choice-video> pip install -r .\requirements.txt
```
